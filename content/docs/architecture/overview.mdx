## Layers

1. App Router UI pages in `src/app`
2. Shared UI components in `src/components`
3. Data layer through Drizzle in `src/db`
4. API handlers in `src/app/api/*/route.ts`

## Request model

- Server Components fetch data directly with Drizzle for page rendering.
- Client Components call internal APIs for user-triggered writes.
- Auth checks for protected writes happen in API handlers via Clerk `auth()`.
