## Primary entities

- `Courses`
- `CourseRating`
- `CourseCredit`
- `Comments`
- `Professors`
- `Users`
- `UserFavorites`
- `Sessions`

## Forum entities

- `forum_posts`
- `forum_comments`
- `forum_notifications`
- `forum_user_favorites`
- `forum_trendy_posts`

## Source of truth files

- Prisma schema: `prisma/schema.prisma`
- Drizzle schema: `src/db/schema.ts`

## Notes

- The codebase currently uses Drizzle for runtime queries.
- Prisma schema is still maintained and useful for schema visibility.
- Aggregate course metrics are maintained in `CourseRating` during comment writes.
