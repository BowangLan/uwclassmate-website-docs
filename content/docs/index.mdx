## What this project is

UWClassmate is a University of Washington student platform focused on:

- Course ratings and course detail discovery
- Comment submission and browsing
- Student org website pages (home, about, activities)
- Authentication and profile experience (Clerk)

## Stack at a glance

- Next.js App Router (`next@16`)
- React + TypeScript
- Tailwind + shadcn/Radix UI
- Clerk authentication
- MySQL with Drizzle ORM
